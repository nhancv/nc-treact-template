# Treact Template

## Tech stack
### Server
- Nodemon
- Typescript
- ExpressJs

### Frontend
- React
- Typescript


## Dev
### Server
```
cd server
npm start
```
Access: http://localhost:4000/

### Frontend
```
cd frontend
npm start
```
Access: http://localhost:3000/

## Build
### Server
```
cd server
npm run build
npm run production
```

### Frontend
```
cd frontend
npm run build
```
